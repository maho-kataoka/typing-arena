# typing-arena
リアルタイム対戦できるタイピングゲーム

将来的には
社内イベントや忘年会で使える、  
**タイピングゲームを中心としたミニゲーム基盤（プロトタイプ）**とすることを目指す

まずは「1人用タイピングゲーム＋管理画面」を完成させることを目標にする。

---

## このプロジェクトについて

このアプリは、イベント進行や司会の負担を減らしつつ、  
参加者全員が同時に楽しめるミニゲームを提供することを目的とした Web アプリ。

忘年会での利用を想定しているが、  
勉強会・研修・社内イベントなどにも転用できる汎用的な構成を目指す。

---

## v0（最初のゴール）

### プレイヤー画面
- タイピング問題を1問表示
- 制限時間内に入力
- 入力完了後にスコア表示
  - 正確性
  - 入力速度（簡易）
- 1人で遊べること

### 管理者画面
- タイピング問題を登録・編集・削除できる
- 問題に難易度を設定できる
- 出題する問題を選択できる

※ チーム戦・同時対戦・観戦画面は v1 以降で対応予定

---

## ゲーム仕様（v0）

### タイピング問題
- 問題文：1〜2行程度のテキスト
- 難易度：easy / normal / hard（点数補正用）
- 制限時間：固定（例：30秒）

### スコア
- 正確性：入力文字数に対する正解率
- 速度：入力完了までの時間を基に算出
- 難易度によってスコアに補正をかける

---

## 技術スタック（予定）

- Frontend / Backend: TypeScript + Node.js
- Framework: Next.js（API Routes 含む）
- Database: SQLite（軽量・ローカル想定）
- ORM: Prisma（予定）
- Hosting: 未定（ローカル or Vercel 想定）

---

## 設計方針

- 小さく作って、確実に完成させる
- 管理画面を最初から含める
- 実装は AI（Claude）を活用し、自分は設計とレビューに集中する
- 将来の拡張（チーム戦・リアルタイム対戦）を見据えた構成にする

---

## 画面構成（v0）

/（プレイヤー画面）
- タイピング問題文を表示
- 入力用テキストフィールド
- 残り時間の表示
- 入力完了後、スコアを表示
- 正確性
- 入力速度
- 再挑戦ボタン（任意）

/admin（管理画面）
- タイピング問題の一覧表示
- 問題の追加
- 問題の編集
- 問題の削除
- 出題する問題の選択

---

## やらないこと（v0）

- 複数チーム同時対戦
- リアルタイム通信（WebSocket）
- 派手なアニメーション
- デザインの作り込み
- 認証・権限管理の厳密化
